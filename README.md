# 📚 Exercícios JavaScript

Projeto criado para praticar lógica de programação com JavaScript puro.  
São **66 exercícios** organizados em cards clicáveis, onde cada um roda direto no navegador — sem precisar abrir o DevTools ou instalar nada.

---

## 🖥️ Preview

> Clique em um card → responda os prompts → veja o resultado na tela.

---

## 🚀 Como usar

1. Faça o download ou clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o arquivo `index.html` no navegador.

3. Pronto! Nenhuma instalação necessária.

---

## 📁 Estrutura de arquivos

```
📂 projeto
 ┣ 📄 index.html   → estrutura da página
 ┣ 📄 style.css    → todo o visual do projeto
 ┗ 📄 main.js      → exercícios + lógica dos cards
```

## 🧠 Conceitos praticados

- Variáveis (`const`, `let`)
- Tipos de dados (string, number, boolean)
- Operadores aritméticos e lógicos
- Condicionais (`if`, `else if`, `else`, `switch`)
- Laços de repetição (`for`, `while`, `do while`)
- Funções
- Arrays e métodos de array (`.filter`, `.find`, `.map`, `.reduce`, `.push`, `.splice`)
- Objetos
- `prompt`, `console.log` e `alert`
